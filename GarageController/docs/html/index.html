Smart Garage Controller

Introduction

The Smart Garage Controller is a modular, extensible, and fully documented embedded automation system built on the Arduino Uno R4 WiFi platform. It provides intelligent control of garage lighting, heating, and door operation using a combination of sensors, relays, and a structured control logic engine.

This documentation describes the complete software architecture, including:

  • Sensor subsystem (DS18B20 temperature, PIR motion, reed switches)
  • Actuator subsystem (relay-driven lights, heater, and door pulse)
  • Control logic (heater hysteresis, door grace timing, lighting automation)
  • User interface (LCD display, menu system, configuration editing)
  • Main application loop and system orchestration

All modules are fully annotated with Doxygen comments and grouped into logical subsystems for clarity and maintainability.

Hardware Overview

The Smart Garage Controller is designed around the following hardware components:

  • Arduino Uno R4 WiFi — main microcontroller and system host
  • 4‑channel 5V relay module — controls lights, heater, and door button
  • DS18B20 temperature sensor — provides temperature readings for heater logic
  • PIR motion sensor (HC‑SR501) — triggers lighting and activity timers
  • Magnetic reed switches (2×) — detect door fully open and fully closed positions
  • 20×4 I²C LCD display — user interface for status and configuration
  • Momentary pushbuttons (3×) — UP / DOWN / SELECT navigation
  • 5V power supply (2–3A) — powers Uno and relay module
  • Screw terminals and wiring — field connections for sensors and relays
  • Project enclosure — protects electronics and provides mounting
