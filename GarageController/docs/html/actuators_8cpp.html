Detailed Description

Implementation of the Actuators subsystem.

This module drives all physical outputs:

  • Lights relay
  • Heater relay
  • Door button relay (timed pulse)

It also tracks the timestamp of the most recent heater relay transition. The Sensors subsystem uses this timestamp to suppress PIR rising edges for a short period after heater relay switching, preventing false motion events caused by electrical noise.
