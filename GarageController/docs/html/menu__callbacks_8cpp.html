Detailed Description

Menu action handlers for the Smart Garage Controller UI.

This module implements the callback functions invoked when the user selects a menu item. These callbacks perform actions such as:

  • Toggling lights
  • Triggering a door pulse
  • Changing heater mode
  • Editing configuration values
  • Exiting the menu

Rendering of menu screens is handled in menus.cpp. UI state transitions are handled in ui_state.cpp.
