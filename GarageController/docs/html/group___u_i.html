◆ pollButtons()

ButtonEvent pollButtons ( )

Polls the physical buttons and returns a debounced event.

  • Returns BTN_NONE if no new event is available.
  • Should be called once per loop() iteration.
Returns
ButtonEvent representing the latest button press, or BTN_NONE.

◆ uiHandleEvent()

void uiHandleEvent ( ButtonEvent ev)

Feeds a button event into the UI state machine.

This function is responsible for:

void uiLoop ( )

Periodic UI processing and LCD refresh.

This should be called once per loop() iteration after control and actuators have been updated. It is responsible for:

  • Updating the current screen
  • Refreshing the LCD at a reasonable rate
