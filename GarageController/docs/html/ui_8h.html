Detailed Description

Public interface for the User Interface (UI) subsystem.

The UI subsystem is responsible for:

  • Handling button input (UP / DOWN / SELECT)
  • Driving the LCD to display status and configuration
  • Managing menus and simple navigation

The main application interacts with the UI only through:

  • ButtonEvent values from pollButtons()
  • uiHandleEvent() to feed button events into the UI state machine
  • uiInit() and uiLoop() for lifecycle management
