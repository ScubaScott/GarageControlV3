Detailed Description

Implementation of the Control Logic subsystem.

This module implements all high-level decision-making for the Smart Garage Controller. It consumes SensorState and produces ActuatorState.

Features:

  • Heater hysteresis control with AUTO/MANUAL/OFF modes
  • Door grace timer allowing heater operation shortly after door opens
  • Motion-based lighting automation with manual override
  • Door pulse request signaling
