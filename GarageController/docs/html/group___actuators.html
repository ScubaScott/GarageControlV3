Applies the desired actuator state to the physical outputs.

  • Drives lights and heater relays directly.
  • Generates a timed pulse for the door relay when requested.

◆ actuatorsLastHeaterChangeMs()

unsigned long actuatorsLastHeaterChangeMs ( )

Returns the timestamp of the last heater relay transition.

The Sensors subsystem uses this timestamp to suppress PIR rising edges for a short period after the heater relay toggles, preventing false motion events caused by electrical noise.

Returns
Milliseconds since boot when the heater relay last changed state.
